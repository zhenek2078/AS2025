# Конвертация файлов journal в текстовые файлы

Включаем через переменную отображение всех строк:

```
SYSTEMD_LESS=FRXMK journalctl
```

```
nano journal.sh
```

```
#!/bin/bash
find /mnt/d/inc -name "*.journal" -print0 | while IFS= read -r -d $'\0' file; do
base_name=$(basename "$file".journal)
output_file="$base_name.txt"
echo "Обработка файла: $file -> $output_file"
journalctl --file "$file" > "$output_file"
if [ $? -ne 0 ]; then
echo "Ошибка: journalctl не выполнился для файла $file"
fi
done
echo "Обработка завершена."
```

```
chmod +x journal.sh
```

```
./journal.sh
```

# Работа с текстом

- cat - вывод содержимого файла на экран;
- less - просмотр содержимого файла постранично;
- head - вывод первых строк файла;

```
head -f /var/log/syslog -n 50 # вывод 50 строк
```

- tail - вывод последних строк файла;

```
tail -f /var/log/syslog # обновление в реальном времени
```

```
tail -f /var/log/syslog -n 50 # обновление в реальном времени, 50 строк
```

- grep - поиск строк в файле;

```
grep -rwn . -e 'flag' # рекурсивынй поиск с выводом номера строки, по вхождению flag
```

```
grep -RiP flag # рекурсивный поиск с игнорированием регистра и регэкспов PERL (можно сложный регэксп, но обычно лень и не требуется)
```

- wc - подсчет количества строк, слов и символов в файле;

```
wc -l # посчет строк
```

```
wc -w # подсчет слов
```

```
wc -m # подсчет символов
```

# Работа с tcpdump и фильтры

Прочитать файл:

```
tcpdump -r rat.pcap
```

С красивым выводом времени, полным отображением и портами в виде цифр:

```
tcpdump -r rat.pcap -tttt -vvvv -n
```

- Фильтр по IP адресу

Пакеты, идущие к или от определенного IP-адреса

```
tcpdump -r rat.pcap host 172.16.1.4
```

Пакеты, идущие от определенного IP-адреса

```
tcpdump -r rat.pcap src host 172.16.1.4
```

Пакеты, идущие к определенному IP-адресу

```
tcpdump -r rat.pcap dst host 172.16.1.4
```

- Фильтр по порту:

Пакеты, идущие к или от определенного порта

```
tcpdump -r rat.pcap port 80
```

Пакеты, идущие от определенного порта

```
tcpdump -r rat.pcap src port 22
```

Пакеты, идущие к определенному порту

```
tcpdump -r rat.pcap dst port 443
```

- Фильтр по протоколу:

Пакеты TCP

```
tcpdump -r rat.pcap tcp
```

Пакеты UDP

```
tcpdump -r rat.pcap udp
```

Пакеты ICMP

```
tcpdump -r rat.pcap icmp
```

извлечь двоичные файлы вредоносного ПО. Команды со скринов:

# Скрипт массового прогона IOC

```
# Скрипт на PowerShell для удобного поиска по результатам утилиты UAC.
# Параметры скрипта
$InputFile = "D:\workdir\input_ioc_this.txt" # Файл, содержащий строки для поиска
$SearchDirectory = "D:\inc\" # Каталог, в котором нужно искать
$OutputFile = "D:\workdir\search_results_ioc_this.txt" # Файл для записи результатов
$LogFile = "D:\workdir\search_log.txt" # Файл для логирования просмотренных файлов

# Чтение строк из входного файла
try {
  $SearchStrings = Get-Content $InputFile
  $StringCount = $SearchStrings.Count # Подсчет количества строк
  Write-Host "Найдено $StringCount строк в файле '$InputFile'."
}
catch {
  Write-Error "Ошибка при чтении файла '$InputFile': $($_.Exception.Message)"
  exit 1
}

# Засекаем время начала
$StartTime = Get-Date

# Функция для поиска строки в файлах
function Search-StringInFiles {
  param (
    [string]$String,
    [string]$Directory
  )

  # Получаем список всех файлов в указанном каталоге и подкаталогах
  $Files = Get-ChildItem -Path $Directory -File -Recurse -Exclude "*.journal"

  foreach ($File in $Files) {
    # Записываем имя просмотренного файла в лог
    "$($File.FullName)" | Out-File -FilePath $LogFile -Append
    # Увеличиваем счетчик просмотренных файлов
    $FilesViewedCount++

    try {
      # Ищем строку в файле
      $SearchResults = Select-String -Path $File.FullName -Pattern $String -Exclude "*.journal"
      if ($SearchResults) {
        # Если строка найдена, записываем результат в файл
        foreach ($Match in $SearchResults) { # Перебираем все совпадения в файле
          $ResultString = "Строка '$String' найдена в файле '$($File.FullName)': '$($Match.Line)'"
          Write-Output $ResultString | Out-File -FilePath $OutputFile -Append
        }
      }
    }
    catch {
      Write-Warning "Ошибка при обработке файла '$($File.FullName)': $($_.Exception.Message)"
    }
  }
}

# Главный цикл: перебираем строки для поиска и ищем их в файлах
Write-Host "Начинаем поиск..."
foreach ($SearchString in $SearchStrings) {
  Write-Host "Ищем строку: '$SearchString'"
  Search-StringInFiles -String $SearchString -Directory $SearchDirectory
}

# Засекаем время окончания
$EndTime = Get-Date

# Вычисляем время работы
$TimeTaken = New-TimeSpan -Start $StartTime -End $EndTime

# Выводим результаты
Write-Host "Поиск завершен. Результаты записаны в файл: '$OutputFile'"
Write-Host "Лог просмотренных файлов записан в файл: '$LogFile'"
Write-Host "Количество просмотренных файлов: $FilesViewedCount" # Вывод количества просмотренных файлов
Write-Host "Время выполнения: $($TimeTaken.ToString())"
```
