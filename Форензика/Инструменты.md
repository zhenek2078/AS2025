# Установка qemu для монтирования дисков

```
sudo apt install qemu-utils
```

# Конвертировать форматы дисков

## VMDK → RAW

```
qemu-img convert -p -f vmdk -O raw disk.vmdk disk.img
```

## VHD / VHDX → RAW

```
qemu-img convert -p -f vhdx -O raw disk.vhdx disk.img
```

## QCOW2 → RAW

```
qemu-img convert -p -f qcow2 -O raw disk.qcow2 disk.img
```

## E01 → RAW

```
ewfexport disk.E01
```

## AUTO-определение (часто работает)

```
qemu-img convert -p -O raw disk.vmdk disk.img
```

# Подключение диска в kali

## vmdk

Подгружаем моудль:

```
sudo modprobe nbd max_part=16
```

Подключаем vmdk:

```
sudo qemu-nbd --read-only --connect=/dev/nbd0 disk.vmdk
```

Смотрим разделы:

```
lsblk /dev/nbd0
```

Монтируем нужный:

```
sudo mount -o ro /dev/nbd0p1 /mnt/forensic
```

Или через loosetup:

```
sudo losetup -r -f -P disk.vmdk
```

```
lsblk
```

```
sudo mount -o ro /dev/loop0p1 /mnt/forensic
```

## dd, img, bin

```
sudo losetup -r -f -P disk.img
```

```
lsblk

```

## Plaso

Установка:

```
mkdir -p /opt/plaso
```

```
cd /opt/plaso
```

```
python3 -m venv plaso
```

```
source plaso/bin/activate
```

```
pip3 install plaso
```

Конвертация:

```
log2timeline --storage-file /root/inc/disk.plaso /root/inc/disk.raw
```

Или сразу сбор:

```
log2timeline.py timeline.plaso /mnt/forensic
```

Сортировка - здесь важно указать верный диапазон дат, чтобы не насобирал кучу лишнего:

```
psort --output-time-zone 'Europe/Kirov' -o l2tcsv -w timeline-mft-trueconf.csv mft-trueconf.plaso "date > datetime('2025-01-01T00:00:00') AND date < datetime('2026-03-25T00:00:00')"
```
