# Поиск файлов по имени и по содержимому

Поиск файла по названию:

```
find / -name 'flag*'
```

Поиск файла по содержимому:

```
grep -rE 'flag[0-9]+\{[^}]+\}' / # Ищем флаги формата flagN{random_string}
```

Или:

```
grep -r "flag*" /
```

Если файл бинарный:

```
grep -r --binary-files=text "flag*" /
```
